About This Project

This project is an extended version of the CP1404 Flask Wikipedia demo.

Features demonstrated in this project include:

  • Using the Flask web framework
  • Creating routes and passing data between pages
  • Using Jinja2 template inheritance
  • Searching Wikipedia using the wikipedia Python library
  • Handling errors such as disambiguation and missing pages
